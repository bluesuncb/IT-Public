Antes de empezarNecesitas un dispositivo con acceso a internet y un navegador web (Chrome, Edge, Safari, Firefox…)Tener Microsoft Authenticator configurado previamente en tu móvil es imprescindible para aprobar la verificación en dos pasos. Si aún no lo has configurado, consulta primero la guía de configuración de Microsoft Authenticator.
1
Acceder al portal de correo
Paso 1.1
Abre el navegador y accede al portal
Abre tu navegador web e introduce la dirección outlook.office.com en la barra de direcciones. Pulsa Enter para acceder.
INICIO DE SESIÓN
2
Inicio de sesión
Paso 2.1
Introduce tu correo electrónico
Escribe tu correo corporativo de BlueSun (@bluesuncb.com) y pulsa Siguiente.
Paso 2.2
Introduce tu contraseña
Introduce tu contraseña corporativa y pulsa Iniciar sesión.
VERIFICACIÓN
3
Verificación MFA
Paso 3.1
Verificación MFA — código de 2 dígitos
El sistema mostrará una pantalla con un código de 2 dígitos. Abre Microsoft Authenticator en tu móvil, introduce el código que se muestra en pantalla y pulsa Sí para aprobar el acceso.
Paso 3.2
Introducir código en el Authenticator
En la pantalla aparecerá un código de 2 dígitos. Abre Microsoft Authenticator en tu móvil, introduce el código que se muestra y pulsa Sí para confirmar.
4
Bandeja de entrada
Paso 4.1
Acceso completado
Una vez aprobado el acceso, verás la bandeja de entrada de tu correo corporativo de BlueSun lista para usar.
ACCESO COMPLETADO ✓
✅
