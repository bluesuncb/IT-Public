ℹ️
Antes de empezarNecesitas un iPhone con iOS 16.0 o superior, conexión a internet y tu correo corporativo de BlueSun (@bluesuncb.com).Tener Microsoft Authenticator configurado previamente en tu dispositivo es imprescindible. Si aún no lo has configurado, consulta primero la Guía de configuración de Microsoft Authenticator para iOS.
1
Descargar aplicación desde el App Store
Paso 1.1
Abre el App Store
Localiza el icono del App Store en tu iPhone y pulsa sobre él para abrirlo.
Paso 1.2
Busca "Microsoft Outlook"
Pulsa en la barra de búsqueda inferior y escribe Microsoft Outlook. El primer resultado será la app de Microsoft Corporation.
Paso 1.3
Pulsa "Obtén" para descargar
Toca el botón Obtén y confirma con Face ID, Touch ID o tu contraseña de Apple ID. La descarga comenzará automáticamente.
💡 La aplicación es completamente gratuita.
INSTALACIÓN
2
Abrir la aplicación Microsoft Outlook
Paso 2.1
Abre Outlook
Localiza el icono Outlook en tu iPhone y pulsa sobre él para abrirlo.
Paso 2.2
Pantalla de bienvenida
Al abrir la app por primera vez verás la pantalla de bienvenida. Pulsa "Agregar cuenta" para continuar.
3
Configurar la cuenta de BlueSun
Paso 3.1
Introduce tu correo electrónico
Escribe tu correo corporativo de BlueSun (@bluesuncb.com) y pulsa Agregar cuenta.
Paso 3.2
Introduce tu contraseña
Introduce tu contraseña corporativa y pulsa Iniciar sesión.
Paso 3.3
Aprobar la verificación MFA
El sistema enviará una notificación push a tu móvil a través de Microsoft Authenticator preguntando si estás intentando iniciar sesión. Pulsa Sí para aprobar el acceso.
⚠️ En ocasiones puede aparecer una pantalla solicitando un código de 2 dígitos en lugar de la notificación. En ese caso, abre Microsoft Authenticator, introduce el código que se muestra en pantalla y pulsa Sí.
Paso 3.4
¿Agregar otra cuenta?
La app preguntará si deseas agregar otra cuenta. Pulsa "Quizás más tarde" para continuar.
Paso 3.5
Cuenta configurada
Outlook mostrará tu bandeja de entrada con el correo corporativo de BlueSun listo para usar.
💡 Es posible que el sistema solicite permiso para enviar notificaciones. Se recomienda activarlas si deseas recibir un aviso cada vez que llegue un nuevo correo corporativo a tu bandeja de entrada.
CONFIGURACIÓN COMPLETADA ✓
✅
¡Listo! Outlook está configurado Ya tienes acceso a tu correo corporativo de BlueSun desde tu iPhone. Recuerda mantener la app actualizada.
